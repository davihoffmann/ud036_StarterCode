# Projeto: Site de trailer de filmes
Trabalho relacionado ao curso de Programa Nanodegree Desenvolvedor Web Full-Stack da Udacity.

##Requerimentos
Python 2.7.x

##Instalação
Faça download do repositório ou faça a clonagem do projeto utilizando o comando:
`git clone https://github.com/davihoffmann/ud036_StarterCode.git`

Abra o diretório onde você fez o download do projeto e execute o arquivo entertainment_center.py

##Adicionar ou Alterasr os daodos dos filmes
Abra o arquivo entertainment_center.py
Crie uma nova instacia da classe Movie, passando como parâmetro o titulo, url do poster e url do trailer do filme
Execute novamente o arquivo entertainment_center.py
